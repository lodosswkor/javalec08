자바의 기초문법 ( C계열 언어 공통 )

1. 주석문 (comments) 
   - 설명문 --> 컴파일러(자바컴파일러)가 해석하면 안되는 설명문
   // 한 줄 주석
   /*   */  여러줄 주석 
   /**  */ 자바 도큐먼트 주석  

2. ; 은 자바 문장(실행문)의 끝 -> ; 없는 문장은 끝나지 않은 문장이다. 
3. 
    a) 자바파일명은 대문자로 시작한다. ( 자바파일명 = 클래스명 ) 
    b) 행위(메서드 = method)의 이름(명)은 소문자로 시작합니다.
